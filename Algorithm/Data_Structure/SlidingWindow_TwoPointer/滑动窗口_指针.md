# 双指针 https://www.acwing.com/blog/content/4788/

- 常用的双指针法有一下几类：
    - 左右指针：两个指针，相向而走，中间相遇。
    - 快慢指针：两个指针，有快有慢，同向而行。
    - 灵活运用：两个指针，灵活运用，伺机而动。
